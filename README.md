# 🚴 Fremont Bridge Cyclist Dashboard

An interactive dashboard built with **Python**, **Streamlit**, **Pandas**, and **Plotly** to explore cyclist traffic across Seattle's Fremont Bridge.

## Features

- 📅 Filter data by Year, Month, and Weekday
- 📊 Interactive line and bar charts
- 🚴 Compare East and West sidewalk cyclist counts
- 📈 Visualise hourly cyclist traffic patterns
- 📸 Fremont Bridge image for additional context

## Technologies Used

- Python
- Streamlit
- Pandas
- Plotly

## Dataset

Seattle Fremont Bridge Bicycle Counter dataset.
